About this model

  • Algorithm: Decision Tree Classifier
  • Dataset: Kaggle Credit Card Fraud 2013
  • Features: PCA-transformed V1–V28 + Time + Amount
  • Train/Test split: 80% / 20%
  • Class 0 = Legitimate, Class 1 = Fraud

Feature guide

  • Time — seconds since first transaction
  • V1–V28 — anonymised PCA components
  • Amount — transaction amount (USD)
  • Typical fraud Amount: low (< $10)
  • Typical fraud V values: large negatives
